공통
· 색상 — MONO 고정: 다크 그레이 배경 + 단일 오렌지 액센트(#ff6b2c). 라이브 콘솔·라운지 스튜디오 양쪽에서 가독 좋음.
· border-radius: 0 — 전역 !important, 라운딩 없음
· EQ — 원형 노브 SVG + 12시 센터 디턴트 틱 + 현재 위치 포인터 + arc
·  HI=+30°(boost) / MID=0°(center) / LO=-20°(cut) 로 mock
· SMPTE — 가로 1행 [MASTER] [A] [B], 높이 28px, CUE/PLAY(34) + STOP(28) + SMPTE(28) 수직 정렬
· VU — 가로 열(column) 배치의 세로 바. HW=[VU][F] 2열, Virtual=[L][R][F] 3열. 페이더 세로 바 복원.
· BPM — 4요소 복원: base 124.00 + +3.23% + cur 128.00 + KEY 8A

레이아웃 차이
· A · CANONICAL — 기존 덱 구조 그대로 (기존 코드 최소 변경). 가장 익숙한 배치, 덱 1개 높이 ~280px.
· B · WAVE-FIRST — 파형 120px 확장, 모든 제어 요소를 하단 단일 행으로. 큐 맞추기·비트 매칭에 최적, ~280px.
· C · TC-HERO — 타임코드 54px 초대형 + BPM pill 오른쪽, 식별·헤더 컴팩트. SMPTE 쇼·리허설·드라이버용, ~340px.
